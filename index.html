WCAG 2.1 Accessibility & Contrast Compliance

A system is only usable if it is accessible. To maintain compliance with strict contrast guidelines, we designate Deep Brown (#501A0D) as our main functional text element on light surfaces. Below is the active math behind our tokens:

Foreground Background Ratio WCAG Level Optimal Application
Deep Brown (#501A0D) Cream Surface (#FAF8F5) 12.5:1 PASSES AAA Primary body copy, documentation tables, and nested inline scripts.
Deep Brown (#501A0D) Lavender Pink (#FBCBFA) 9.4:1 PASSES AAA Highlight markers, system cards, and active navigation controls.
Deep Brown (#501A0D) Soft Salmon (#F99B86) 6.5:1 PASSES AAA (Large) / AA (Body) Static metadata layouts, warning borders, and tags.
Deep Brown (#501A0D) Muted Mustard (#D19D14) 5.8:1 PASSES AA Primary sidebar elements, warning cards, and metadata.
Cream Light (#FAF8F5) Vibrant Orange (#F95417) 3.1:1 PASSES AA (Large Text Only) Large display components and titles (minimum 18pt/24px bold).
Compliance Rule: Vibrant Orange (#F95417) is strictly kept as a design accent border or as a background block wrapped behind Deep Brown (#501A0D) text. It is never used for general body copy, ensuring an elegant user journey across all interfaces.
